Job details

Rooms

Each room holds the objects being painted — add walls, ceilings, doors etc as needed. Each object carries its own dimensions, count, paint type # × coats, and primer type # × coats (0 = no primer). Two doors with different paint? Add two door objects. The description field beside each type is the same field as in Paint Needed & Purchase — fill it in while you measure and it shows in both places.

Paint needed & purchase

This section is the output of the rooms above. Each paint category can hold multiple paint types (distinct products/colours). Referencing a type # on any object creates its line here automatically, and every object across every room using the same type # accumulates into that one line. Coats are set per object — the same paint can get 1 coat in one room and 3 in another. NEEDED = each object's area × its coats, summed per type. Fill in tin size, tins bought, and price beneath it to satisfy it, and it'll tell you if you're buying enough.
